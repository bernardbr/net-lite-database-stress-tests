# HOW TO RUN

UPGRADE YOUR PIP
`py -m pip install --upgrade pip`

INSTALL VIRTUALENV
`py -m pip install --user virtualenv`

CREATE VIRTUALENV
`py -m venv env`

ACTIVE THIS
`.\env\Scripts\activate`

INSTALL REQUIREMENTS
`pip install -r requirements.txt`

RUN LOCUST
`locust`

ACCESS YOUR [BROWSER](http://localhost:8089/) AND ENJOY!
